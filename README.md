    <p><strong>Time needed:</strong> 3-4 hours</p>
    
    <div class="step">

        <p><strong>What to do:</strong> Weigh olivine (100 mg or 500 mg per bottle), spread thin in Petri dish, UV for 30 minutes, use immediately</p>
